About Fabian's Library

Welcome to Fabian's Library
We provide a place for those of you who want to improve, explore, and gain more experience about life.
This library was built on 1966 by Mr Fabian, in which this library name implies.
Mr Fabian was a great inventor also professor on 20th century. He built this library because on that era, there are so many people who want to learn and learn some books, yet didn't have any money to buy a book. So Mr Fabian tried to solve this problem, by built a charge-free library called “The Library”, so people could enjoy learn something new. In 1994, Mr Fabian died and the library was run by his younger sister named Mrs Sarah. To commemorate his amazing services to society, his sister and all of the library’s board decided to change the name of the library to “Fabian’s Library”. Until this time, Fabian’s Library is very famous all around the world, including Singapore, China, Indonesia, Brazil, and many more.
To keep up with globalization, Mrs Sarah decided to make an online library, so people all around the world can access Fabian’s Library more easily and also to commemorate Mr Fabian’s great services to society. So feel free to enjoy and explore more about life in this library!
